blimputils.magnetometer

magnetometer.py

Module for the BMM350 magnetometer. Provides class infrastructure and implementation of underlying methods for interacting with the sensor.

This module defines classes for sensor data structures, the main Magnetometer class for sensor interaction, and an I2C implementation Magnetometer. It relies on magnetometer_vars.py for sensor-specific constants and registers. Calibration is handled using user-provided hard and soft iron matrices and the local Earth’s magnetic field strength.

class blimputils.magnetometer.Magnetometer(bus: int, addr: int = 20)

Bases: _MagnetometerBase

I2C interface implementation for the BMM350 Magnetometer. This class handles the specifics of I2C communication and inherits core magnetometer functionalities from _MagnetometerBase.

Parameters:

  • bus (int) – The I2C bus number (e.g., 1 for Raspberry Pi’s /dev/i2c-1).

  • addr (int, optional) – The I2C address of the BMM350 sensor. Defaults to BMM350_I2C_ADDR_DEFAULT (0x14).

Raises:

  • IOError – If the I2C bus cannot be opened or the platform is not supported.

  • TypeError – If bus or addr are not integers.
