blimputils.accelerometer

Accelerometer module for the blimp-utils library, specifically for the Bosch-Sensortec BMI270.

class blimputils.accelerometer.Accelerometer(bus: int, addr: int = 104)

Bases: object

A class to represent the BMI270 accelerometer.

Parameters:

  • bus (int) – The I2C bus number (e.g., 1 for Raspberry Pi).

  • addr (int) – The I2C address of the accelerometer (typically 0x68 or 0x69 for BMI270).
