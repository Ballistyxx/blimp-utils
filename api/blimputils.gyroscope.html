blimputils.gyroscope

Gyroscope module for the blimp-utils library, specifically for the Bosch-Sensortec BMI270.

class blimputils.gyroscope.Gyroscope(bus: int, addr: int = 104, default_odr: int = 8, default_bwp: int = 2, default_range: int = 0)

Bases: object

A class to represent the BMI270 gyroscope.

Parameters:

  • bus (int) – The I2C bus number (e.g., 1 for Raspberry Pi).

  • addr (int) – The I2C address of the BMI270 (hardwired to 0x68).

  • default_odr (int) – Default Output Data Rate for the gyroscope, e.g., GYR_ODR_100.

  • default_bwp (int) – Default Bandwidth Parameter for the gyroscope, e.g., GYR_BWP_NORMAL.

  • default_range (int) – Default range setting for the gyroscope, e.g., GYR_RANGE_2000.

Raises:

  • TypeError – if bus or addr are not integers.

  • IOError – if I2C bus cannot be opened or BMI270 initialization fails.

get_x_raw() → int

Get the raw X-axis gyroscope value from BMI270. Uses GYR_X_7_0 (LSB) and GYR_X_15_8 (MSB) registers.

Returns:

The raw X-axis gyroscope value.

Return type:

int

Example:

raw_x = gyro.get_x_raw()
